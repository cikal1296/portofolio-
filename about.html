My Skill

Html (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membangun struktur dasar dari sebuah halaman web. Elemen-elemen HTML, seperti div, p, h1, dan a, digunakan untuk menampilkan teks, gambar, tautan, dan bagian lainnya di halaman web. HTML bertindak sebagai fondasi dari semua elemen visual di situs web.


Css (Cascading Style Sheets)

CSS adalah bahasa stylesheet yang digunakan untuk mendesain tampilan dan tata letak halaman web. Dengan CSS, Anda dapat mengatur warna, font, margin, padding, tata letak, dan animasi elemen-elemen HTML. CSS memungkinkan pengembang untuk memisahkan konten (HTML) dari desain, sehingga situs web menjadi lebih fleksibel dan mudah diatur.


JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan fungsionalitas dinamis ke halaman web. Dengan JavaScript, Anda dapat membuat elemen yang responsif, seperti form yang memvalidasi input, tombol interaktif, animasi, dan banyak lagi. JavaScript bekerja di sisi klien (browser) dan merupakan salah satu pilar utama pengembangan web modern bersama HTML dan CSS.


framework

node.js adalah libray JavaScript yang digunakan untuk membangun user interface yang interaktif berbasis component.
